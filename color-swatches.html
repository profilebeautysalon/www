Button CSS Implementation

.btn-primary {
    background-color: var(--primary-color);
    color: white;
}

/* For higher contrast buttons */
.btn-primary.high-contrast {
    background-color: #e87f6c; /* Darker version of primary color */
    color: white;
}

.btn-secondary {
    background-color: var(--secondary-color);
    color: white;
}

.btn-secondary.high-contrast {
    background-color: #5c4538; /* Darker version of secondary color */
    color: white;
}

.btn-outline {
    background-color: transparent;
    border: 2px solid var(--primary-color);
    color: var(--primary-color);
}

.btn-outline:hover {
    background-color: var(--primary-color);
    color: white;
}

How to Implement

To use any of these color schemes, replace the current CSS variable values with the selected palette in your CSS file:

:root {
    /* Colors */
    --primary-color: #5abfad;         /* Example: Teal Coastal Primary */
    --primary-light: #f3d8cb;         /* Keeping the original Primary Light */
    --secondary-color: #1a7b6a;       /* Example: Teal Coastal Secondary */
    --accent-color: #dd8e6c;          /* Example: Teal Coastal Accent */
    --dark-color: #1a3c38;            /* Example: Teal Coastal Dark */
    --light-color: #f0f7f6;           /* Example: Teal Coastal Light */
    --text-color: #333333;            /* Adjusted text color */
    --text-light: #666666;            /* Adjusted light text color */
    --bg-color: #fff9f8;              /* Slightly adjusted background color */
    --border-color: #e0e0e0;          /* Adjusted border color */
}
